# Git Sync & Remote

Commands to synchronize source code with the server (GitHub/GitLab).

## 1. Push code to server
```bash
git push origin <branch-name>
```

## 2. Pull latest code
```bash
git pull origin main
```

## 3. Update information from remote
```bash
git fetch origin
```

## 4. Check remote URL
```bash
git remote -v
```
