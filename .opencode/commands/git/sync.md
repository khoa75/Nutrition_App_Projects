# Git Sync & Remote

Các lệnh để đồng bộ mã nguồn với server (GitHub/GitLab).

## 1. Đẩy code lên server
```bash
git push origin <branch-name>
```

## 2. Tải code mới về
```bash
git pull origin main
```

## 3. Cập nhật thông tin từ remote
```bash
git fetch origin
```

## 4. Kiểm tra remote URL
```bash
git remote -v
```
