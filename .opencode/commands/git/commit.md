# Git Commit & Staging

Commands to save changes into the repository's history.

## 1. Move files to the Staging Area
```bash
git add .
```
- Adds all current changes.

## 2. Save Changes (Commit)
```bash
git commit -m "US-XX: Brief description of changes"
```

## Commit Message Rules
- Use clear English.
- Attach the User Story or Task ID (e.g., US-1).
- Concisely describe the purpose of the commit.
