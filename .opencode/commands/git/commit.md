# Git Commit & Staging

Các lệnh để lưu lại các thay đổi vào lịch sử của repository.

## 1. Đưa file vào khu vực chờ (Staging Area)
```bash
git add .
```
- Thêm toàn bộ các thay đổi hiện tại.

## 2. Lưu thay đổi (Commit)
```bash
git commit -m "US-XX: Brief description of changes"
```

## Quy Tắc Commit Message
- Sử dụng tiếng Anh hoặc tiếng Việt rõ ràng.
- Gắn ID của User Story hoặc Task (VD: US-1).
- Mô tả súc tích mục đích của commit.
