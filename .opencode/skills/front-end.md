# Skill: Front-end Development (Flutter & React)

Skills for building smooth and highly interactive user interfaces.

## 1. Flutter (Mobile) Expertise
- **State Management**: Effectively use `Provider` or `Riverpod` for global state management.
- **Clean Architecture**: Separate UI logic and business logic through Repository and UseCase layers.
- **Animations**: Use `Implicit Animations` or `Lottie` to enhance user experience.

## 2. React (Web Admin) Expertise
- **Hooks Architecture**: Maximize use of `useMemo` and `useCallback` to optimize render performance.
- **Component Design System**: Build highly reusable components.
- **Form Management**: Use `React Hook Form` or `Formik` to handle complex forms.

## 3. Data Visualization
- **Charts**: Use `fl_chart` (Flutter) and `Recharts` (React) to visualize nutritional and weight data.
