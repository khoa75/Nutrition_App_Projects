# Skill: Front-end Development (Flutter & React)

Kỹ năng xây dựng giao diện người dùng mượt mà và tương tác cao.

## 1. Flutter (Mobile) Expertise
- **State Management**: Sử dụng `Provider` hoặc `Riverpod` để quản lý state toàn cục một cách hiệu quả.
- **Clean Architecture**: Tách biệt logic UI và logic nghiệp vụ thông qua các lớp Repository và UseCase.
- **Animations**: Sử dụng `Implicit Animations` hoặc `Lottie` để tăng trải nghiệm người dùng.

## 2. React (Web Admin) Expertise
- **Hooks Architecture**: Tận dụng tối đa `useMemo`, `useCallback` để tối ưu hóa hiệu năng render.
- **Component Design System**: Xây dựng các component có khả năng tái sử dụng cao (Reusable Components).
- **Form Management**: Sử dụng `React Hook Form` hoặc `Formik` để xử lý các biểu mẫu phức tạp.

## 3. Data Visualization
- **Charts**: Sử dụng `fl_chart` (Flutter) và `Recharts` (React) để trực quan hóa dữ liệu dinh dưỡng và cân nặng.
