# Skill: DevOps & Cloud Infrastructure

Skills to operate, deploy, and ensure the system maintains high stability (Uptime ≥ 99.5%) in accordance with project constraints.

## 1. Containerization & Orchestration
- **Docker**: Containerize services (Spring Boot, FastAPI, React) into independent containers to ensure consistency across dev, staging, and production environments.
- **Docker Compose**: Manage and easily launch the entire technology stack in the local environment.

## 2. CI/CD & Automation
- **GitHub Actions**: Build pipelines to automate the linting, testing, building, and deployment processes.
- **Automated Backups**: Set up cronjobs or cloud schedules to automatically back up data on a daily basis.

## 3. Cloud Deployment (AWS / GCP)
- **Load Balancing**: Configure load balancing systems to route traffic, designing an architecture capable of supporting a minimum of 100,000+ active users.
- **Monitoring & Logging**: Implement centralized resource tracking (CPU/RAM) and logging tools to promptly detect and resolve errors, ensuring API response times are always < 2 seconds.
