# Project Glossary

This document defines the key terms (Ubiquitous Language) used consistently across all modules (Backend, Frontend, AI, Admin) to avoid confusion.

## 1. Users & Permissions
- **User**: Regular mobile application user.
- **Admin**: Administrator using the React Admin Dashboard.
- **Account / UserProfile**: Detailed user information (height, weight, BMI, nutritional goals).

## 2. Nutrition System
- **Food / FoodItem**: An independent type of food or dish (e.g., Apple, Bowl of Pho).
- **Meal**: A meal during the day (Breakfast, Lunch, Dinner, Snack). A Meal can contain multiple FoodItems.
- **DietLog / MealRecord**: Food diary recorded by the user during the day.
- **NutritionFacts**: Nutritional information table (Calories, Protein, Fat, Carbs).
- **BMI (Body Mass Index)**: Body Mass Index.

## 3. AI System
- **Scan / FoodAnalysis**: The process of AI recognizing food from an image.
- **ConfidenceScore**: The AI's confidence level when predicting a dish (usually from 0.0 to 1.0).
- **Recommendation**: Personalized menus or nutritional advice suggested by the system.
