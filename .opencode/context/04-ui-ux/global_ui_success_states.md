# UI Component Spec: Success Actions & Feedback Toasts
*   Platform: Multi-platform Context (Mobile & Web Grid)
*   Theme: Highly visible positive UI feedback indicators - Light Mode compatible

## Component 1: Floating Success Toast Notification
*   Container Frame: Capsule-shaped rounded horizontal card (Width up to 320dp, height 48dp, corner radius 8px).
*   Visual Parameters: Background solid Green (#4CAF50), soft outer elevation drop shadow (Shadow-2).
*   Internal Layout Matrix (Left-to-Right):
    1. Icon: Green Circle with White Checkmark tick asset (20dp size, aligned left).
    2. Text Message: "Meal logged successfully!" (Inter 14px Medium font, solid White color text).
*   Layout Position: Floating at the Bottom-Center for Mobile view, or Bottom-Right corner for Web View.

## Component 2: Gamified Achievement Pop-up Modal (Mobile)
*   Container Box: Centered 12px Rounded Pop-up Card layout (#FFFFFF Surface, crisp shadow elevation).
*   Visual Graphic Header: Vector illustration asset of colorful celebration confetti explosions surrounding a golden trophy emblem.
*   Text Content Group:
    *   Header: Title "Daily Goal Reached! 🌟" (Inter 22px Bold, Accent Yellow/Gold #FFB400 text color).
    *   Body text description: "Awesome job! You hit your target protein and calorie budget for today perfectly." (Text color #1E293B).
*   Call To Action: "Keep It Up" primary full-width mint green button wrapper at the base (#2ED5C5, text #1E293B).
