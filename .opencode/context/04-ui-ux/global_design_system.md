# UI-UX Design System Foundations

## 🎨 Color Palettes

### 🌐 Client App (Mobile) - Clean Light Vibe
*   Background: #F5F5F5 (Light Gray)
*   Surface/Cards: #FFFFFF (Pure White)
*   Borders/Dividers: #E2E8F0 (Light Slate Border)
*   Primary Accent: #2ED5C5 (Mint Green)
*   Secondary Accent: #3B82F6 (Electric Blue)
*   Warm Accent: #FFB400 (Sunshine Yellow)
*   Text Primary: #1E293B (Dark Slate text)
*   Text Secondary: #64748B (Medium Muted Gray)

### 🖥️ Admin Dashboard (Web) - Professional Light Vibe
*   Background: #FAFAFA (Off-White)
*   Surface/Cards: #FFFFFF (Pure White)
*   Borders: #E0E0E0 (Light Border)
*   Primary Accent: #76FF03 (Lime Green)
*   Secondary Accent: #303F9F (Deep Indigo)
*   Text Primary: #212121 (Dark Gray)
*   Text Secondary: #757575 (Medium Gray)

## 📐 Typography & Layout Base
*   Font Family: Inter & Outfit (Sans-Serif)
*   Spacing Grid: 8-point system (8px, 16px, 24px, 32px padding/margins)
*   Corner Radius: 8px for standard cards, 12px for modals, 4px for buttons
*   Elevation: Shadow-1 (0 1px 3px rgba(0,0,0,0.12)), Shadow-2 (0 4px 6px rgba(0,0,0,0.15))
*   Icons: Feather Icons style (Outline 2px stroke)
